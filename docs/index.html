Features

  • Panel-first UX with quick actions and templates.
  • TSV → Interlinear (gb4e) with spacing controls.
  • TSV ↔ LaTeX tabularx conversions.
  • LaTeX tabularx → Excel export.
  • Image import → ready-to-compile LaTeX figure block.

See full details in the README.

Requirements

  • Recommended: LaTeX Workshop
  • Recommended: LaTeX Utilities

Settings highlights

  • Tables dir: Default output for LaTeX tables.
  • Figures dir: Default output for imported images.
  • Main TeX file: Makes figure paths relative.
  • Excel export location: Downloads, Documents, Workspace, or Prompt.
  • Interlinear spacing: `\\openup` amount for gloss lines.
